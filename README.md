# Code Review

This repository reflects how I approach code reviews from both a quality and security perspective. The goal is not just to check if code works, but to identify bugs, evaluate maintainability, assess potential risks, and provide clear, actionable feedback.

In addition to traditional reviews, this repo also includes examples of evaluating and ranking AI-generated code and technical responses. This includes looking at things like correctness, clarity, reasoning, and overall usefulness.

---

## Areas Covered

* Python
* SQL
* JavaScript
* Security-focused code reviews
* AI-generated code evaluation
* Technical response assessment

---

## Skills Demonstrated

* Bug and logic flaw identification

* AI response evaluation and ranking
* Recommendations and remediation guidance
